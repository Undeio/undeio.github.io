WEB 服务器

IP 地址概要

发表于 2021-08-15 分类于 网络安全

IP 地址

  1. 路由器用户连接外网和内网；
  2. 交换机用于组建局域网（内网）；
  3. IP 由 4 个 8 位二进制组成，通常会转换成点分十进制，方便记忆；
  4. 局域网的通讯规则：所有 IP 必须处于同一网段才可以互相通讯；
  5. IP 地址的构成：网络位 + 主机位；

虚拟机的分类

发表于 2021-08-08 分类于 网络安全

虚拟机按照运行架构可以分为两类；

原生架构

  1. 虚拟机作为应用软件安装在操作系统上；
  2. 可以在此应用软件上安装多个操作系统；
  3. 直接安装在硬件上的操作系统作为宿主；

例如 VMware Fusion，VMware Workstation，Parallels Desktop，Oracle VM VirtualBox；

寄居架构

  1. 虚拟机软件直接安装在计算机硬件上；
  2. 虚拟机本身就是一个操作系统；
  3. 在这个虚拟机中可以同时运行多个操作系统；

例如 VMware vSphere；
