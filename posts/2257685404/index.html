Sublime Text 手动注册激活

发表于 2022-05-18 分类于 macOS

汇总 Sublime Text 手动注册激活的方法，方法来源于互联网；

Sublime Text 3

此方法适用于 Sublime Text 3 & 4 Alpha 4094；

  1. 拷贝sublime_text(Linux & macOS)或sublime_text.exe(Windows)的副本至其他位置：

    • Linux 默认位于/opt/sublime_text/sublime_text；
    • macOS 默认位于/Applications/Sublime Text.app/Contents/MacOS/sublime_text；
    • Windows 默认位于C:\Program Files\Sublime Text\sublime_text.exe；

  2. 在 hexed.it 中选择并打开sublime_text*，搜索97 94 0D，替换为00 00 00；

  3. 文件另存为 xxx 至你想保存的位置；

  4. 类*unix系统请赋予执行权限chmod +x xxx；

  5. 拷贝并替换安装目录下的源文件；

  7. 注册激活完成；
