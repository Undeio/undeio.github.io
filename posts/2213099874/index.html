IP 地址概要

发表于 2021-08-15 分类于 网络安全

IP 地址

  1. 路由器用户连接外网和内网；
  2. 交换机用于组建局域网（内网）；
  3. IP 由 4 个 8 位二进制组成，通常会转换成点分十进制，方便记忆；
  4. 局域网的通讯规则：所有 IP 必须处于同一网段才可以互相通讯；
  5. IP 地址的构成：网络位 + 主机位；

子网掩码

  1. 子网掩码用于确认网络位：与 255 对应的是网络位，与 0 对应的是主机位；

    例如：

    IP 地址子网掩码网络位主机位
    192.168.0.1255.255.255.0192.168.0.1
    192.168.0.1255.255.0.0192.168.0.1
    192.168.0.1255.0.0.0192.168.0.1

  2. 每个 IP 都必须有对应的子网掩码；

  3. IP 地址的分类：

    IP 地址分为 5 大类，以 IP 的第 1 位进行区分：

    1. A 类：1 ~ 126，默认子网掩码：255.0.0.0；
    2. B 类：128 ~ 191，默认子网掩码：255.255.0.0；
    3. C 类：192 ~ 223，默认子网掩码：255.255.255.0；
    4. D 类：224 ~ 239，组播地址；
    5. E 类：240 ~ 254，科研使用；
    6. A、B、C 三类也叫单播地址，一个 IP 只能对应 1 台设备；
    7. 255.255.255.255 表示全局广播地址；
    8. 255.x.x.x 用于子网掩码；
    9. 127.0.0.1 表示回环地址，永远代表主机自身；

  4. IP 的主机位不能全为 0，也不能全为 255：

    当主机位全部置 0 时，表示该网段的名称；

    当主机位全部置 255 时，表示该网段的广播地址；

    所以，主机位的可用地址范围是 1 ~ 254；

    例如：

    IP 地址子网掩码网段总称网段广播地址网段可用 IP 范围
    192.168.0.1255.255.255.0192.168.0.0192.168.0.255192.168.0.1 ~ 192.168.0.254
    192.168.0.1255.255.0.0192.168.0.0192.168.255.255192.168.0.1 ~ 192.168.255.254
    192.168.0.1255.0.0.0192.0.0.0192.255.255.255192.0.0.1 ~ 192.255.255.254

网关

  1. 网关用于连接交换机和路由器；
  2. 通常，网关 IP 是网段的首地址或最后一个地址，且为固定 IP；

使用 ping 测试网络连通性

有去有回才算成功通讯；

  1. ping 目标 IP 地址：测试网络连通性；
  2. ping -t 目标 IP 地址：一直 ping；
  3. ping -n 数字 目标 IP 地址：修改 ping 包的数量；
  4. ping -l 数字 目标 IP 地址：修改 ping 包的大小；
